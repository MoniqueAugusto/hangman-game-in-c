# Hangman Game in C

Hangman game developed in C language for a Computer Science first semester project.

## How to Run

Compile the program:

```bash
gcc forca.c -o forca
```

Run the program:

```bash
./forca
```

## Features

- Random words and categories
- Input validation
- ASCII art hangman
- Repeated letter verification
- Victory and defeat screens
- Terminal interaction
